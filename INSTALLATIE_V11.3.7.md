# Installatie Planyx PWA v11.3.7

Na een geslaagde GitHub Pages-deployment:

1. Open Planyx in Safari op de iPhone.
2. Tik op **Delen**.
3. Kies **Zet op beginscherm**.
4. Controleer dat de naam **Planyx** wordt getoond.
5. Tik op **Voeg toe**.

Verwijder een eerder beginschermicoon eerst wanneer iOS nog een oud logo of een oude naam toont. Voor deze release is geen Supabase-actie nodig.
