# Installatie v11.3.4

1. Pak `Planning-GJsystems-v11.3.4-GitHub.zip` uit.
2. Upload de volledige inhoud naar de hoofdmap van GitHub-repository `Planningv2` en overschrijf de bestaande bestanden.
3. Wacht tot de GitHub Action volledig groen is.
4. Open de app eenmaal met `?v=113400` om de vernieuwde app-cache direct te laden.
5. Test op laptop en iPhone vanuit Historie bij een uitgevoerd bezoek de knop `PDF`.

Voor v11.3.4 hoeft geen nieuwe SQL in Supabase te worden uitgevoerd.
