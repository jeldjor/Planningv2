# Installatie Planyx v11.3.6

1. Upload de volledige inhoud van deze ZIP naar de hoofdmap van de GitHub-repository.
2. Vervang de bestaande bestanden wanneer GitHub daarom vraagt.
3. Wacht tot de GitHub Pages-workflow groen is.
4. Open de app eenmaal met `?v=113600` achter de URL.
5. Sluit op iPhone het oude tabblad volledig en open de app opnieuw.

Voor deze merkwijziging is geen SQL, Edge Function of andere Supabase-actie nodig.
